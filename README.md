# Real Time Object Tracker

Детекция объектов на видео

# Клонирование репозитория

```shell
git clone https://github.com/rntprch/real-time-obj-tracker.git
cd real-time-obj-tracker
```

# Установка

Установка и активация виртуального окружения:

```shell
python3 -m venv .venv
source .venv/bin/activate        # Linux/macOS
.venv\Scripts\activate           # Windows
```

Установка зависимостей

```shell
pip install -r requirements.txt
```

# Запуск инференса

```
python3 entrypoint.py --input_path crowd.mp4 --output_path output.mp4
```

# Выводы по работе

Код успешно выполняет поставленную задачу: на видеоролике корректно находятся люди и поверх кадра отображаются рамки с указанием уверенности модели. Большинство объектов класса "person" в кадре распознаны правильно.

## Анализ качества распознавания

- Модель хорошо находит людей на средней дистанции и в центральной части кадра.
- На дальнем плане и при частичных перекрытиях возможны пропуски.
- В условиях плотной толпы некоторые объекты сливаются, и модель снижает уверенность.
- Ложных срабатываний немного благодаря фильтрации только класса "person".

## Как улучшить качество

- Использовать более крупную модель YOLOv8 (l или x).
- Дообучить модель на датасетах по детекции людей: CrowdHuman, MOT17/20, CityPersons.
- Применить трекинг с постобработкой для стабильных рамок.
- Повысить разрешение входного видео или применить super-resolution.
